Read City
Учебная информационная система автоматизации книжного магазина: каталог товаров, оформление заказов с самовывозом и панель управления заказами для персонала.
Реализована в виде десктопного WPF-клиента и отдельного REST API, работающих с общей базой данных MS SQL Server.
ReadCity.DataAccess - общая бизнес-логика и доступ к данным. Используется и десктоп-клиентом, и веб-API, что исключает дублирование кода.
ReadCity.Api - тонкий слой контроллеров ASP.NET Core, открывающий бизнес-логику по HTTP с автодокументацией Swagger.
ReadCity.Desktop - клиентское приложение на WPF для покупателей и персонала магазина.

Функциональность
- Каталог товаров (доступен без авторизации)
- Просмотр полного списка книг с фото, автором, описанием, ценой, скидкой и остатком на складе.
- Поиск по наименованию (без учёта регистра).
- Фильтрация по производителю (издательству) и диапазону цен.
- Сортировка по наименованию или цене, по возрастанию/убыванию.
- Счётчик «показано N из M» товаров.
Корзина и оформление заказа
- Добавление товаров в текущий заказ прямо из каталога.
- Просмотр корзины с возможностью изменить количество или удалить позицию; итоговая сумма пересчитывается автоматически.
- Заказ можно оформить как авторизованным пользователем, так и гостем.
- При оформлении заказу присваивается уникальный номер и случайный трёхзначный код получения.
- Формирование и сохранение в файл (.txt) талона на получение заказа с полным составом, суммой и кодом получения.
Авторизация и роли
- Вход по логину/паролю; сессия сохраняется в настройках и восстанавливается при следующем запуске приложения.
- Три роли: Администратор, Менеджер, Авторизированный клиент.
- Функции управления заказами доступны только администратору и менеджеру.
- Управление заказами (для менеджера/администратора)
- Поиск заказа по номеру.
- Просмотр сведений о заказе: дата, клиент.
- Изменение даты доставки и статуса заказа.
Веб-API
- Тот же набор бизнес-операций (авторизация, товары, заказы, справочники), доступный через REST для интеграции с другими клиентами.
- Интерактивная документация Swagger UI.
